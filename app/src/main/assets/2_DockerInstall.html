Установка Docker в Ubuntu 19.04

Загрузка и установка ключа авторизации пакетов:

curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -

Добавить репозитарий Docker в Ubuntu 19.04:
add-apt-repository "deb [arch=amd64] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable edge"
apt-get update
apt-cache policy docker-ce

Если не выводит ничего, то добавляем другой (тестовый) репозитарий Docker:

add-apt-repository "deb [arch=amd64] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable edge test"

Обновляем базу пакетов:
apt-get update

Проверяем наличие пакетов в обновленной базе:
apt-cache policy docker-ce

Должна отобразиться информация о кандидате на установку:

docker-ce:

Candidate: 5:19.03.0~1.4.beta4-0~ubuntu-disco

…..
Производим установку:
apt-get install docker-ce

После завершения установки проверяем статус работы сервиса:
systemctl status docker

Проверяем версию:
docker --version

Запускаем тестовый контейнер Docker:

Docker version 19.03.0-beta4, build e4666ebe81
docker run hello-world

При успешном тесте должно отобразиться:

Hello from Docker!
