Docker Hub

Docker Hub является крупнейшим в мире хранилищем изображений контейнеров с массивом источников контента, включая разработчиков сообщества контейнеров, проекты с открытым исходным кодом и независимых поставщиков программного обеспечения (ISV), которые создают и распространяют свой код в контейнерах. Пользователи получают доступ к бесплатным общедоступным репозиториям для хранения и обмена изображениями или могут выбрать план подписки для частных репозиториев.


Частные хранилища для обмена изображениями контейнеров

Docker Hub - это служба репозитория, предоставляемая Docker для поиска и обмена изображениями контейнеров с вашей командой. Ключевые особенности включают в себя:

    Частные репозитории: Push и Pull изображения контейнера
    Автоматизированные сборки: автоматически создавать образы контейнеров из GitHub и Bitbucket и отправлять их в Docker Hub.
    Команды и организации: Управление доступом к частным репозиториям
    Официальные изображения: извлекайте и используйте высококачественные изображения контейнеров, предоставленные Docker
    Изображения издателя: извлекайте и используйте высококачественные изображения контейнеров, предоставленные сторонними поставщиками. Сертифицированные образы также включают поддержку и гарантию совместимости с Docker Enterprise

    Webhooks: запускать действия после успешной отправки в репозиторий для интеграции Docker Hub с другими сервисами
